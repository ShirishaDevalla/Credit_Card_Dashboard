# Credit_Card_Dashboard
## Project Objective
This dashboard is developed using Microsoft Excel and Power Query to analyze and visualize customer demographic and financial attributes. It consolidates raw customer data into an interactive and structured format, enabling users to explore key variables.

## Dataset Used
- <a href="https://github.com/ShirishaDevalla/Credit_Card_Dashboard/blob/main/CreditCardData.csv">Dataset</a>

## Question (KPI's)
- Which education level has the highest number of customers?
- How does marital status impact customer distribution?
- Which card category (Blue, Silver, Gold, Platinum) has the most customers?
- How are card categories distributed across different income groups?
- Which income group has the highest number of dependents?
- Is there a relationship between marital status and income level?
- What is the relationship between number of dependents and income category?
- How does income category vary with the number of dependents?

## Skills
- Excel
- power query
- Data Management

## Process
Imported raw customer data into Excel from the source file.
Removed unnecessary columns and corrected data types.
Handled missing and inconsistent values.
Organized cleaned data into a structured table for analysis.
Created relationships and calculated fields where required.
Built Pivot Tables and Charts for key metrics.
Added slicers for interactive filtering (age, income, card category, etc.).
Designed a clean and user-friendly dashboard layout.

## conclusion
This dashboard effectively presents a comprehensive view of customer demographics and financial characteristics using Excel and Power Query. By analyzing factors such as age, education, marital status, number of dependents, income category, and card type, the dashboard reveals clear patterns in customer behavior and segmentation. The insights show that higher income and education levels are generally associated with premium card categories (Gold and Platinum), while customers with lower income levels are more concentrated in Blue and Silver cards.
